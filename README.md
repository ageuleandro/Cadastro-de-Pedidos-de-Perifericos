# Cadastro de Pedidos de Periféricos
Sistema simples para gerenciar pedidos de periféricos.

## Estrutura
- **bin/start.rb**: Arquivo principal para iniciar o sistema.
- **lib/**: Contém o código principal.
- **spec/**: Contém testes (se aplicável).
- **config/**: Arquivos de configuração.

## Como usar
1. Clone o repositório.
2. Execute o sistema: `ruby bin/start.rb`.
